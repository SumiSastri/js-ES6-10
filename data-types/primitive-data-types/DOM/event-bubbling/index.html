JAVASCRIPT EVENT BUBBLING DEMO

If you click circle1 (in the console) shows circle1 has been clicked
If you click circle2 or circle3 you will only see circle1
When you click circle2 in the console you will see both circle1 and circle2 clicks
When you click circle3 in the console you will see circle3, circle2 & circle1 cliks
If you change the innerHTML all three circles only the innerHTML of circle1 is changed

Event bubbling is the way browsers read or parse an event handler

click body div -> click html root div -> click document object -> window object
