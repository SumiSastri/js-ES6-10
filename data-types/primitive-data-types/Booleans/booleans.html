ES-6 Iterables, Iteration & Booleans

Open the console for a demo

In programming languages you need a data type that will take only one of 2 answers - yes or no/ true or false / on or off.

These help execute functions in programming. Everything with a value returns true. There are certain default boolean values - anything without a value returns false.

Booleans are especially useful in iterables - a loop will continue to match true and stop executing if false, so a control can be written based on these boolean values expected.

ES-6 has 2 new protocols - the Iterable and the Iterator protocols.Iterable allows JavaScript Objects to define and customise their own behavior.Iterator is a standard way to produce a sequence of values in a collection.
