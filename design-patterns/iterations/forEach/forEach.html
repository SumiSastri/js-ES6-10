ES-6 forEach

Syntax: array.forEach(callbackFunction(element, index, array), thisValue){return function body with logic}

Call the function array.forEach(callbackFunctionName)

the value of this the second param in the callback is optional

Example of how this syntax is written out with an example

  • let numbers = [1, 2, 3, 4];
  • function updateNumbers(elementValue, i, numbers) { return numbers[i] = elementValue * 10; }
  • numbers.forEach(updateNumbers)

    • The function is passed by reference and called as a param of the forEach method
